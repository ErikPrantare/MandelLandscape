To build, run:
```
git clone git@github.com:ErikPrantare/MandelLandscape.git
cd MandelLandscape
git checkout develop
cmake . -Bbuild
cmake --build ./build/
mv ./build/src/MandelLandscape .
```

If everything built correctly, the program can now be run from `./MandelLandscape`
